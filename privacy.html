4. Принципы обработки данных

Оператор руководствуется следующими принципами (Art. 5 GDPR / ст. 5 ФЗ-152 / ст. 5 Закона РК о персональных данных):

  • Законность и добросовестность — обработка осуществляется на законном основании, добросовестно и прозрачно
  • Минимизация данных — мы собираем только те данные, которые необходимы для заявленных целей
  • Ограничение целей — данные обрабатываются только для целей, указанных в настоящей Политике; обработка в иных целях не допускается без дополнительного согласия
  • Точность — мы принимаем разумные меры для актуализации и исправления данных
  • Ограничение хранения — данные хранятся не дольше, чем необходимо для достижения целей обработки
  • Целостность и конфиденциальность — обеспечиваем защиту данных техническими и организационными мерами
  • Подотчётность — Оператор готов продемонстрировать соблюдение настоящей Политики

Субъект данных имеет право:
